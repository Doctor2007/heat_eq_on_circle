# Heat Equation on a Circle

This project explores the numerical solution of the heat equation on irregular 2D domains, specifically a disk (circle) and a donut shape. The simulation is implemented in Python and visualised using Jupyter Notebooks.

## Overview
- **Domains:** 2D unit square, and 2D circular/donut regions.
- **Core Files:** 
  - `andrii_summative2_aqm.ipynb`: Solves the 2D heat equation, simulating heat diffusion over time on a disk and a donut using domain masking.

## Getting Started

### Prerequisites
Make sure you have Python installed. You can install the required dependencies using the provided `requirements.txt`:

```bash
pip install -r requirements.txt
```

### Usage
1. Open the project folder in VS Code or Jupyter Notebook.
2. Open `andrii_summative2_aqm.ipynb`.
3. Run the cells sequentially. 
> *Note:* Some high-resolution grid simulations in the second notebook might take a while to compute. It is recommended to click "Run All" and read through the markdown derivations as the visuals generate.
